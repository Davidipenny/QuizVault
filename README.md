# QuizVault — 通用刷题系统

适用于各学科选择题复习的桌面应用。支持单选题、多选题和判断题，提供错题本、收藏夹、进度保存等功能。

内置哈尔滨工业大学毛概题库作为示例，可自行添加任意学科的题库。

## 下载使用

**无需安装 Python，下载解压即用。**

前往 [GitHub Releases](https://github.com/Davidipenny/QuizVault/releases/latest) 下载 `QuizVault-GUI.zip`，解压后双击 `quiz_gui.exe` 即可。

## 添加自己的题库

程序启动时会自动扫描 `banks/` 文件夹，每个子文件夹就是一个独立题库。

**三步添加新题库：**

1. 在 `banks/` 下新建文件夹，例如 `banks/马原/`
2. 将题目文件（`.md` 格式）放入该文件夹
3. 重启程序，新题库自动出现在列表中

同一文件夹下的多个文件会自动合并为一个题库。

### 题目格式

用 Markdown 编写，格式如下：

**判断题：**

```markdown
## 判断题

**1. 实践是检验真理的唯一标准（　）**
A. 正确
B. 错误

**答案：A**

**解析：** 这是马克思主义认识论的基本观点。
```

**单选题：**

```markdown
## 单选题

**1. 马克思主义最鲜明的特征是（　）**
A. 科学性
B. 实践性
C. 革命性
D. 人民性

**答案：B**

**解析：** 实践性是马克思主义最鲜明的特征。
```

**多选题：**

```markdown
## 多选题

**1. 以下属于唯物辩证法的有（　）**
A. 对立统一规律
B. 质量互变规律
C. 否定之否定规律
D. 因果规律

**答案：ABC**

**解析：** 唯物辩证法三大规律为对立统一、质量互变、否定之否定。
```

- 判断题答案为 A 或 B
- 单选题答案为 A-D 中的一个字母
- 多选题答案为 2-4 个不重复字母
- 判断题必须有 2 个选项，单选/多选题必须有 4 个选项
- 用 `---` 分隔每道题
- 完整格式规范见 [docs/FORMAT.md](docs/FORMAT.md)

### 用 AI 生成题目

将上方格式示例粘贴给 AI（ChatGPT、Claude 等），加上指令如：

> 请按上面的 Markdown 格式，生成 [科目名] 的选择题 50 道。

生成后保存为 `.md` 文件，放入 `banks/` 对应文件夹即可。

## 功能一览

| 功能 | 说明 |
|------|------|
| 刷题模式 | 顺序/随机，按题型（判断/单选/多选/全部）筛选 |
| 错题本 | 答错自动记录，答对自动移除，支持错题重练 |
| 收藏夹 | 收藏重要题目，可创建多个收藏夹，支持导出 |
| 标记题目 | 刷题时按 `F` 标记有疑问的题目，后续统一处理 |
| 进度保存 | 中途退出自动保存，下次可继续或重新开始 |
| 批量删题 | 勾选式批量删除不需要的题目 |
| 快捷键 | A/B/C/D 选择，Enter 提交，Space 下一题，F 标记 |

## 常见问题

**Q: exe 打不开？**
A: 可能被杀毒软件拦截，请添加信任后重试。需要 Windows 10 及以上系统。

**Q: 如何添加更多题目？**
A: 在 `banks/` 下新建文件夹，放入 `.md` 格式的题目文件，重启即可。也可用 AI 生成（见上方模板）。

**Q: 如何重置错题本？**
A: 删除题库文件夹内的 `wrong_questions.json` 文件即可。

**Q: 刷题进度保存在哪？**
A: 在题库文件夹内的 `quiz_progress.json`，答完全部题目后自动清除。

**Q: 导入时提示"重复题目"？**
A: 题目基于内容去重。用 `--force` 参数可强制覆盖：`python import_questions.py import --force 文件.md`

## 从源码运行

需要 Python 3.6+，无强制外部依赖。

```bash
# GUI 版本
python gui/main.py

# 导入题目
python import_questions.py import 题目.md

# 构建 exe
pyinstaller gui.spec
```

## 文件结构

```

├── gui/                  # GUI 版本（唯一活跃版本）
│   ├── main.py           # 入口
│   ├── pages/            # 各页面模块
│   └── widgets/          # 可复用组件
├── bank_manager.py       # 题库管理模块
├── import_questions.py   # 题库导入/导出工具
├── parse_markdown.py     # Markdown 解析器
├── banks/                # 题库数据
│   └── 毛概往年题/       # 示例题库
├── gui.spec              # GUI 打包配置
├── legacy/               # 已废弃的终端版本
└── docs/FORMAT.md        # 题库格式规范
```
